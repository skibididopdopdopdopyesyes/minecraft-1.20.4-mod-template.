# Minecraft 1.20.4 Mod Template

Базовый шаблон мода для Minecraft 1.20.4 с использованием Forge.

## Сборка

1. Клонируйте репозиторий: `git clone <url>`
2. Откройте в IDE (IntelliJ IDEA или Eclipse) как Gradle‑проект
3. Выполните `gradlew build` в терминале
4. Готовый файл `.jar` будет в папке `build/libs/`
